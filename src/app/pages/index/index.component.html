Descripción del sitio

Este sitio web utiliza a la OMDb API para mostrar información sobre películas. Al inicio de cada página, se presenta un menú de navegación para acceder rápidamente a diferentes páginas y secciones. En la pestaña "Inicio" se muestran los campos del dataset y algunos registros de este en cuadrículas para mostrar póster, título, fecha de lanzamiento, director, tiempo de duración y categoría de la película. En la pestaña "Datos" se muestra una tabla con opciones para ordenar y filtrar las columnas Year, Runtime e imdbRating y debajo de esta se muestran gráficos para visualizar los datos. Por último, en la pestaña "Contacto" se muestra la información de contacto y un formulario para enviar comentarios o preguntas. En el pie de página se proporcionan enlaces a políticas de privacidad, términos de servicio, datos de copyright y atribuciones

¿Qué es OMDb API?

OMDb API se refiere a la Open Movie Database Application Programming Interface (Interfaz de Programación de Aplicaciones de la Base de Datos de Películas Abierta). La OMDb API es una API gratuita y pública que proporciona acceso a una extensa base de datos de información sobre películas y programas de televisión.
